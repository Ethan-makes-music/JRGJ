# About Continuance 
Continuance is a game made for The Very Serious Juniper Dev Game Jam

You find a magical clock in a world you just woke up in. And that clock can spin back or forward time

You go around completing quests people ask you to do, while trying to find out why you are there in the first place

Have fun with the source code here!

## Controls:

- WASD or Arrow Keys for player

- E to Interact

- Q to time travel
